# Allen J. Elite Jet Concierge

Premium private aviation brokerage platform connecting verified private jet owners/operators with elite individuals seeking charter services.

## 🌐 Live Site

**https://allen-j-elite-jet.vercel.app**

## 📋 Pages

1. **Home** - Hero landing with services overview
2. **Client Registration** - Multi-step KYC registration
3. **Login** - Client and admin authentication
4. **Flight Request** - Quote request form
5. **Aircraft Showcase** - Fleet gallery
6. **Quote Center** - Pricing breakdown
7. **Payment** - Multiple payment methods
8. **Travel Itinerary** - Journey timeline
9. **Concierge** - Premium services
10. **Client Dashboard** - Account hub
11. **Admin Panel** - Fleet & pricing management

## 🛠 Tech Stack

- **Framework:** Next.js 14
- **Runtime:** React 18
- **Deployment:** Vercel
- **Version Control:** GitHub
- **Styling:** Inline CSS (no dependencies)

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000)
